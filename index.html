Tip 2: Know Your Selectors

Fastest to slowest selectors are:

  • id selectors
  • element selectors
  • class selectors
  • pseudo & attr selectors (no browser-based call to take advantage of)

ID and element are faster as backed by native DOM operations

Es muy importante conocer como trabajan los selectores para poder ir llevando acabo mejoras de rendimiento o para utilizar las opción con mejor rendimiento directamente en cada caso
